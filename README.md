# Rasha_Do

Rasha_Do is a simple task manager app built with Flutter.

## Description

Rasha_Do is a simple and smart task manager app built with Flutter.
It helps users organize their daily tasks efficiently,
allowing them to add, edit, delete, and mark tasks as completed.
The app supports task prioritization and due dates,
making task management easy and intuitive.

## Features
- Add, edit, delete tasks
- Set task priority (Low, Medium, High)
- Set due date for tasks
- Show completed and pending tasks
